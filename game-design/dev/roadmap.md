# 开发路线图

> **版本:** 0.1 - 骨架
> **状态:** 待填充

---

## 一、版本规划总览

```
Phase 1: 垂直切片 (Vertical Slice)
    │
    ├── 核心玩法验证
    ├── 技术架构验证
    └── 1个完整区域 + 1个Boss
    
    ↓

Phase 2: 核心内容 (Core Content)
    │
    ├── 3个区域完整实现
    ├── 核心技能系统
    └── 可玩Demo
    
    ↓

Phase 3: 完整内容 (Full Content)
    │
    ├── 全部区域
    ├── 全部技能
    └── Early Access
    
    ↓

Phase 4: 打磨与发布 (Polish & Release)
    │
    ├── 平衡性调整
    ├── Bug修复
    └── 正式发布
```

---

## 二、Phase 1: 垂直切片

### 目标
- 验证核心战斗手感
- 验证技能系统基础
- 验证关卡设计流程
- 产出可展示的Demo片段

### 范围

| 内容 | 详情 | 状态 |
|------|------|------|
| 区域 | 初始墓地（完整） | 待开发 |
| 敌人 | 骷髅战士（2-3种变体） | 待开发 |
| Boss | 墓地守卫 | 待开发 |
| 技能 | 翻滚、钩索、轻/重攻击 | 待开发 |
| 装备 | 初始武器、钩索护符 | 待开发 |
| UI | 基础HUD、菜单 | 待开发 |

### 预计周期
- 2-4周（独立开发者）

---

## 三、Phase 2: 核心内容

### 目标
- 构建可发布的Demo
- 验证技能组合系统
- 建立内容生产流水线

### 范围

| 内容 | 详情 | 状态 |
|------|------|------|
| 区域 | 腐朽森林、沉没教堂、灰烬矿坑 | 待开发 |
| Boss | 腐朽之母、堕落主教、矿坑蠕虫 | 待开发 |
| 技能 | 灰烬回响、圣光术、震地击 | 待开发 |
| 系统 | 技能组合、升级树、装备强化 | 待开发 |
| NPC | 灰烬主城NPC（铁匠、商人、先知） | 待开发 |

### 预计周期
- 4-8周
