## 一、整体风格定义

### 风格定位
**Dark Fantasy Pixel Art** — 黑暗幻想像素艺术

### 关键词
```
Dark fantasy pixel art, 16-bit retro style, gloomy medieval atmosphere, 
muted desaturated colors with warm accent lighting, high contrast shadows, 
hand-crafted pixel detail
```

### 风格参考

| 参考游戏 | 借鉴点 |
|---------|--------|
| Blasphemous | 哥特式阴暗、宗教意象 |
| Dead Cells | 流畅动画、清晰可读性 |
| Hollow Knight | 氛围感、精细细节 |
| Salt and Sanctuary | 魂系气质的像素化 |

## 二、色彩方案

### 主色调

| 色彩角色 | 色值范围 | 用途 |
|---------|---------|------|
| **背景暗色** | #1a1a2e ~ #2d2d44 | 环境背景、阴影 |
| **中间调** | #4a4a6a ~ #6b6b8d | 建筑、地面、中景 |
| **前景亮色** | #8b8baa ~ #aaaaaa | 角色、前景物体 |

### 强调色

| 色彩角色 | 色值 | 用途 |
|---------|------|------|
| **火光暖色** | #e6a23c ~ #f0c060 | 篝火、火把、温暖光源 |
| **危险红** | #c0392b ~ #e74c3c | 血迹、伤害提示、危险 |
| **魔力蓝** | #2980b9 ~ #3498db | 技能特效、魔法元素 |
| **毒/腐绿** | #27ae60 ~ #2ecc71 | 毒素、自然腐化 |
| **诅咒紫** | #8e44ad ~ #9b59b6 | 诅咒、深渊、暗影 |

### 色彩使用原则
1. **暗色为主，亮色点缀** — 70%暗色，20%中间调，10%强调色
2. **光源自发光** — 火光、技能光是唯一的暖色来源
3. **对比度优先** — 角色必须与背景有足够对比，确保可读性

## 三、分辨率与尺寸

### 角色尺寸

| 角色类型 | 尺寸 | 说明 |
|---------|------|------|
| 玩家角色 | 64x64 | 标准尺寸 |
| 普通敌人 | 64x64 | 与玩家同级 |
| 精英敌人 | 64x64 ~ 128x128 | 稍大以示区分 |
| Boss | 128x128 ~ 256x256 | 大尺寸，压迫感 |

### 环境尺寸

| 元素类型 | 尺寸 | 说明 |
|---------|------|------|
| 地形瓦片 | 32x32 或 64x64 | 可平铺 |
| 道具 | 32x32 ~ 64x64 | 根据重要性 |
| 背景 | 纵向1080px | 视差滚动 |

### UI尺寸

| 元素类型 | 尺寸 | 说明 |
|---------|------|------|
| 图标 | 32x32 | 背包、技能栏 |
| 血条外框 | 256x32 | HUD |
| 菜单 | 自适应 | 全屏或半屏 |

## 四、动画规范

### 帧率

| 动作类型 | 帧数 | 说明 |
|---------|------|------|
| 待机 | 4-6帧 | 循环 |
| 行走 | 6-8帧 | 循环 |
| 奔跑 | 6-8帧 | 循环 |
| 轻攻击 | 4-6帧 | 单次 |
| 重攻击 | 6-8帧 | 单次 |
| 翻滚 | 4-6帧 | 单次 |
| 受击 | 2-3帧 | 单次 |
| 死亡 | 6-8帧 | 单次 |

### 动画原则
1. **关键帧优先** — 起手、打击点、收手必须清晰
2. **可读性 > 流畅性** — 玩家能看清动作比流畅更重要
3. **帧数服务于手感** — 快速动作用少帧，慢速动作用多帧

## 五、特效规范

### 特效类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 攻击特效 | 武器轨迹、命中火花 | 斩击、刺击 |
| 技能特效 | 技能释放的视觉反馈 | 钩索、残影、血誓 |
| 环境特效 | 场景氛围 | 灰烬飘落、雾气、火焰 |
| 状态特效 | 状态异常表现 | 流血、中毒、燃烧 |

### 特效原则
1. **不遮挡游戏性** — 特效不能挡住玩家需要看的东西
2. **层次分明** — 前景特效、中景特效、背景特效分开处理
3. **适度夸张** — 像素风允许适度夸张，但不要过度
