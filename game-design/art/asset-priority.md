# 素材优先级

> **版本:** 0.1 - 骨架

## 优先级定义

| 等级 | 说明 | 何时需要 |
|------|------|---------|
| **P0** | 核心素材，没有就无法开发 | 垂直切片 |
| **P1** | 重要素材，影响体验 | Alpha |
| **P2** | 标准素材，提升品质 | Beta |
| **P3** | 锦上添花，可后续补充 | 正式版 |
