## 一、核心设计理念

### 魂系战斗哲学
- **有规律可循** — 每个敌人的攻击都有模式，可以学习
- **高风险高回报** — 贪刀会死，耐心输出有奖励
- **死亡是学费** — 每次死亡都学到东西，不是惩罚
- **多种解法** — 同一个敌人可以用不同策略应对

## 三、完整数值系统

### 3.1 玩家基础属性

| 属性 | 基础值 | 每点收益 | 说明 |
|------|--------|---------|------|
| 生命 (HP) | 500 | +25 | 最大生命值 |
| 专注 (FP) | 100 | +3 | 技能消耗 |
| 体力 (Stamina) | 100 | +5 | 翻滚/攻击消耗 |
| 力量 | 10 | +3%物理伤害 | 物理伤害加成 |
| 敏捷 | 10 | +2%攻速/翻滚距离 | 速度相关 |
| 信仰 | 10 | +3%技能效果 | 技能相关 |

### 3.2 伤害计算公式

```
基础伤害 = 武器基础伤害 × (1 + 力量% + 武器补正%)
技能伤害 = 基础伤害 × 技能倍率 × (1 + 信仰%)
最终伤害 = 技能伤害 × 属性克制倍率 × (1 - 防御减伤%)
暴击伤害 = 最终伤害 × 1.5（暴击率由敏捷决定）
```

**属性克制表：**

| 攻击属性 | 被克制属性 | 克制倍率 |
|---------|-----------|---------|
| 火焰 | 自然/冰霜 | 1.5 |
| 圣光 | 暗影/不死 | 1.5 |
| 暗影 | 圣光 | 1.3 |
| 物理 | 无 | 1.0 |

**防御减伤公式：**
```
防御减伤% = 敌人防御 / (敌人防御 + 100)
```

### 3.3 体力系统

| 操作 | 体力消耗 | 说明 |
|------|---------|------|
| 轻攻击 | 15/次 | 3连击共45 |
| 重攻击 | 30/次 | 单次高伤 |
| 翻滚 | 25/次 | 无敌帧位移 |
| 格挡 | 持续消耗 | 每秒消耗10 |
| 弹反 | 0 | 成功弹反不消耗 |

**体力恢复：**
- 脱战状态：30/秒
- 战斗状态：15/秒
- 体力耗尽惩罚：体力归零后3秒内无法恢复，且无法翻滚/格挡
