# 成长与进度系统

> **版本:** 1.0 - 完整版
> **状态:** 已完成

---

## 一、成长维度

玩家角色有三个成长维度：

| 维度 | 说明 | 来源 |
|------|------|------|
| **属性成长** | 基础数值提升 | 升级 |
| **技能成长** | 技能获取与强化 | 探索/升级 |
| **装备成长** | 装备获取与强化 | 探索/购买/锻造 |

---

## 二、升级系统

### 灰烬精华（经验/货币）

**获取来源：**

| 来源 | 数量 | 说明 |
|------|------|------|
| 普通敌人 | 10-30 | 根据敌人类型 |
| 精英敌人 | 50-100 | 根据敌人类型 |
| Boss | 200-1000 | 根据Boss难度 |
| 灵魂道具 | 100-500 | 使用后获得 |
| 探索奖励 | 50-200 | 隐藏区域 |
| 支线任务 | 100-300 | 完成任务 |

**消耗用途：**

| 用途 | 消耗 | 说明 |
|------|------|------|
| 属性升级 | 递增 | 每级所需精华递增 |
| 技能升级 | 100-400 | 根据升级等级 |
| 装备强化 | 200-1000 | 根据强化等级 |
| 商店购买 | 按价格 | 购买物品 |

**死亡惩罚：**
- 死亡后失去所有灰烬精华
- 精华留在死亡地点
- 再次死亡则原有精华消失
- 成功回收则拿回精华

## 三、篝火系统（存档点）

### 功能详情

| 功能 | 说明 | 消耗 | 条件 |
|------|------|------|------|
| 存档 | 保存游戏进度 | 无 | 自动 |
| 回复 | 恢复所有HP/FP/道具 | 无 | 使用篝火 |
| 升级 | 提升属性 | 灰烬精华 | 有足够精华 |
| 技能 | 更换技能组合 | 无 | 有新技能 |
| 传送 | 在篝火间快速移动 | 无 | 已激活的篝火 |
| 锻造 | 装备强化/修理 | 精华+材料 | 有足够材料 |
| 商店 | 购买物品 | 精华 | 有足够精华 |

### 篝火回复规则
- 使用篝火后，回复所有HP/FP
- 回复所有消耗品（原素瓶等）
- **但是**：所有普通敌人重生
- Boss不会重生
- 已获得的物品不会消失

| 沉没教堂 | 3 | 入口、大厅、Boss房前 |
| 灰烬矿坑 | 3 | 入口、矿井、Boss房前 |
| 灰烬主城 | 1 | 中心广场 |
| 王城废墟 | 3 | 入口、王座厅前、Boss房前 |

## 四、死亡与惩罚

### 死亡流程

```
玩家死亡
    ↓
在最后篝火处复活
    ↓
失去所有灰烬精华
    ↓
精华留在死亡地点（金色光球）
    ↓
玩家前往回收
    ↓
成功回收 → 拿回精华
再次死亡 → 原有精华消失，新精华留在新死亡点
```

### 设计意图
- 死亡有代价（需要回去回收）
- 但不是毁灭性的（有机会拿回）
- 制造紧张感（距离篝火越远风险越大）
- 鼓励谨慎探索（不要盲目冲入危险区域）

### 特殊情况
- 如果死亡地点在Boss房内，精华会留在Boss房门口
- 如果死亡地点在悬崖下，精华会留在悬崖边缘
- 如果死亡地点在毒气区域，精华会留在安全区域

## 五、进度解锁

### 区域解锁顺序

**主线流程（线性）：**
```
初始墓地 → [三选一] → 灰烬主城 → [三选一] → 灰烬之心
```

**可选顺序：**
- 腐朽森林、沉没教堂、灰烬矿坑：可任意顺序
- 王城废墟、深渊回廊、火焰神殿：可任意顺序

**解锁条件：**
| 区域 | 解锁条件 |
|------|---------|
| 腐朽森林 | 击败墓地守卫 |
| 沉没教堂 | 击败墓地守卫 |
| 灰烬矿坑 | 击败墓地守卫 |
| 灰烬主城 | 击败任意一个中期Boss |
| 王城废墟 | 到达灰烬主城 |
| 深渊回廊 | 到达灰烬主城 |
| 火焰神殿 | 到达灰烬主城 |
| 灰烬之心 | 击败所有中期Boss |

### 技能驱动的进度

```
获得新技能
    ↓
回到之前的区域
    ↓
发现之前无法到达的地方
    ↓
获得奖励/发现隐藏内容
```

**技能回访奖励表：**

| 技能 | 可回访区域 | 新发现 |
|------|-----------|--------|
| 钩索 | 所有区域 | 高处平台、远程机关 |
| 灰烬回响 | 腐朽森林、沉没教堂 | 残影机关、隐藏通道 |
| 圣光术 | 沉没教堂、深渊回廊 | 黑暗区域、隐藏文献 |
| 震地击 | 灰烬矿坑、王城废墟 | 脆弱地面、隐藏洞穴 |
| 血誓之刃 | 王城废墟、灰烬之心 | 血祭机关、隐藏宝库 |
| 暗影步 | 深渊回廊、火焰神殿 | 栅栏、隐藏墙壁 |
| 火焰印记 | 火焰神殿、灰烬之心 | 可燃物、火焰机关 |

## 六、Build多样性

### Build类型

| Build | 主要属性 | 核心技能 | 玩法风格 |
|-------|---------|---------|---------|
| 力量型 | 力量、生命 | 震地击、血誓之刃 | 高伤害、高生命、慢速 |
| 敏捷型 | 敏捷、体力 | 暗影步、钩索 | 高攻速、高机动、低生命 |
| 信仰型 | 信仰、生命 | 圣光术、火焰印记 | 技能伤害、治疗、辅助 |
| 均衡型 | 全属性 | 全技能 | 适应性强、无明显短板 |

### Build搭配建议

**力量型Build：**
- 武器：大剑、战斧
- 护符：铁壁、吸血本能
- 技能：震地击、血誓之刃、翻滚
- 玩法：高伤害输出，用生命换取伤害

**敏捷型Build：**
- 武器：匕首、长枪
- 护符：暗影闪避、灰烬亲和
- 技能：暗影步、钩索、翻滚
- 玩法：快速攻击，灵活闪避

**信仰型Build：**
- 武器：法杖、直剑
- 护符：灰烬亲和、残影大师
- 技能：圣光术、火焰印记、灰烬回响
- 玩法：技能输出，辅助队友
