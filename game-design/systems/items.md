# 装备与物品系统

> **版本:** 1.0 - 完整版
> **状态:** 已完成

---

## 一、装备系统概述

### 设计理念
- 装备不只是数值，每件装备**自带技能**
- 换装备 = 换玩法
- 没有绝对的"最强装备"，只有最适合Build的搭配
- 装备外观会改变玩家角色的视觉效果

---

| 槽位 | 数量 | 说明 | 提供内容 |
|------|------|------|---------|
| 主武器 | 1 | 决定基础攻击方式 | 主动技能1、被动特性 |

| 护符 | 2 | 提供特殊效果 | 被动特性、主动技能3 |

| 手套 | 1 | 防御+特殊效果 | 防御值、攻击加成 |
| 靴子 | 1 | 防御+移动相关 | 防御值、移动速度 |

### 武器属性详解

| 属性 | 说明 | 计算方式 |
|------|------|---------|
| 基础伤害 | 固定伤害值 | 直接使用 |
| 攻击速度 | 影响DPS | 基础速度 × 武器速度倍率 |
| 补正 | 属性加成比例 | 基础伤害 × (1 + 属性% × 补正系数) |
| 附加效果 | 流血/中毒/火焰等 | 累积值/持续伤害 |
| 自带技能 | 装备后获得的主动技能 | 消耗专注值释放 |
| 自带特性 | 装备后获得的被动特性 | 满足条件自动触发 |

**补正系数：**
| 补正等级 | 系数 |
|---------|------|
| S | 1.5 |
| A | 1.2 |
| B | 1.0 |
| C | 0.8 |
| D | 0.5 |
| E | 0.3 |
