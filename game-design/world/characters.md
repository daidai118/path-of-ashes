## 一、玩家角色

### 身份
玩家是一个**无名者**——在灰烬中苏醒的不死人，失去了大部分记忆。只记得自己的使命：到达灰烬之心，寻找终结灰烬灾难的方法。随着游戏推进，玩家会发现自己与初火之王有着神秘的联系。

### 外观
- 默认外观：穿着破旧的灰色斗篷，兜帽遮住面容
- 斗篷下是被灰烬侵蚀的身体，皮肤呈现灰色纹理
- 眼睛偶尔闪烁着琥珀色光芒（灰烬之力的象征）
- 外观会根据装备改变，但始终保持"被灰烬侵蚀"的特征

### 背景故事
千年前，初火之王用自身的生命封印了世界核心。封印维持了千年，但如今正在瓦解。玩家是"灰化者"——被灰烬侵蚀但没有完全失去意识的少数存在。他们被一种神秘的力量唤醒，踏上前往灰烬之心的旅程。

随着探索，玩家会发现自己其实是初火之王的"碎片"——王的灵魂分散成多个碎片，每个碎片都承载着王的一部分记忆和力量。这就是为什么玩家能够使用灰烬之力，也是为什么玩家能够到达灰烬之心。

## 三、Boss

### 设计原则
- 每个Boss测试玩家对**特定技能**的理解
- Boss有多个阶段，难度递增
- 死亡后快速重试，减少挫败感
- 每个Boss都有背景故事，通过环境叙事和NPC对话揭示

---

### 3.1 墓地守卫 (The Grave Warden)
| 属性 | 值 |
|------|-----|
| **所在区域** | 初始墓地 |
| **难度** | ★☆☆☆☆ |
| **测试技能** | 翻滚、基础攻击 |
| **设计意图** | 教学Boss，建立信心 |
| **阶段数** | 2 |

**外观描述：**
- 身高2.5倍于玩家的巨大骑士
- 穿着锈蚀的板甲，盔甲上有红色符文发出微光
- 头戴封闭式头盔，眼睛处有红色光芒
- 手持巨大的双手剑，剑身有裂纹和锈迹

**背景故事：**
曾经是守卫墓地的忠诚骑士，在灰烬降临后仍然坚守岗位。但灰烬侵蚀了他的心智，现在他只是本能地阻止任何人进入或离开墓地。击败他后，玩家可以获得墓地钥匙，离开初始区域。

**攻击模式：**
- 第一阶段：横扫、突刺、跳劈（缓慢，有规律）
- 第二阶段：连续横扫、冲刺、怒吼（速度加快，新增组合攻击）

**战斗场地：**
- 圆形竞技场，直径约15格
- 中央有石碑可阻挡冲刺
- 边缘有矮墙，可阻挡部分攻击

**掉落物：**
- 灰烬精华 x200
- 墓地守卫大剑 (10%)
- 墓地钥匙 (100%)

- 不规则森林空地

- 中央有腐朽之树（Boss本体的一部分）

**掉落物：**
- 灰烬精华 x350
- 腐朽之母权杖 (10%)

- 四周有彩色玻璃窗，战斗时会破碎

**掉落物：**
- 灰烬精华 x500

- 第一阶段：火焰斩击、火焰吐息、火焰护盾
- 第二阶段：火焰爆发、连续火焰斩、火焰领域
- 第三阶段：火焰狂暴、全场火焰、火焰治愈

**战斗场地：**
- 火焰神殿大厅，有火把可以点亮
- 地面有火焰区域，进入会累积燃烧值
- 四周有冰霜区域，可以用来灭火

**掉落物：**
- 灰烬精华 x600
- 火焰守护者巨剑 (10%)
- 火焰之证 (100%，用于学习火焰印记技能)

### 3.8 最终Boss：灰烬之王 (The Ash King)
| 属性 | 值 |
|------|-----|
| **所在区域** | 灰烬之心 |
| **难度** | ★★★★★ |
| **测试技能** | 全部技能 |
| **设计意图** | 终极考验 |
| **阶段数** | 5 |

**外观描述：**
- 第一阶段：巨大的灰烬巨人，由灰烬和岩石组成
- 第二阶段：灰烬巨人分裂成多个小灰烬
- 第三阶段：小灰烬重新组合成更强大的形态
- 第四阶段：灰烬之王现出真身——初火之王的碎片
- 第五阶段：灰烬之王与世界核心融合，成为最终形态

**背景故事：**
灰烬之王是初火之王的另一个碎片。当王牺牲自己封印世界核心时，他的灵魂分裂成多个碎片。玩家是希望的碎片，灰烬之王是绝望的碎片。他相信灰烬是世界的终结，而玩家相信灰烬可以被净化。

**攻击模式：**
- 第一阶段：灰烬冲击、灰烬风暴、灰烬召唤
- 第二阶段：多重攻击、灰烬分身、灰烬领域
- 第三阶段：灰烬狂暴、全场灰烬、灰烬治愈
- 第四阶段：初火之力、火焰攻击、黑暗攻击
- 第五阶段：世界核心之力、毁灭性攻击、时间限制

**战斗场地：**
- 灰烬之心，巨大的圆形竞技场
- 地面有灰烬流动，会随机爆发
- 四周有世界核心的碎片，可以互动

**掉落物：**
- 灰烬精华 x1000
- 灰烬之王武器 (100%)
- 世界核心碎片 (100%，用于触发结局)

## 四、隐藏Boss

### 灰烬之影 (The Shadow of Ash)
| 属性 | 值 |
|------|-----|
| **所在区域** | 灰烬之心（隐藏） |
| **难度** | ★★★★★+ |
| **测试技能** | 全部技能+隐藏组合 |
| **设计意图** | 真正的终极挑战 |
| **阶段数** | 3 |
| **触发条件** | 收集所有记忆碎片 |

- 玩家的镜像，但完全被灰烬侵蚀
- 眼睛发出暗红色光芒，表情冷漠
- 使用与玩家相同的技能，但更强大

**背景故事：**
灰烬之影是玩家的"另一面"——如果玩家选择拥抱灰烬而非净化灰烬，就会变成这样。他是玩家可能成为的样子，是玩家必须战胜的"自我"。

**攻击模式：**
- 第一阶段：模仿玩家的技能组合
- 第二阶段：使用隐藏组合技能
- 第三阶段：使用所有技能的强化版本

**掉落物：**
- 灰烬精华 x2000
- 灰烬之影武器 (100%)
- 真结局钥匙 (100%)
