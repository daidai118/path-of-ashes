## 二、区域详情

### 2.1 初始墓地 (The Starting Grave)
| 属性 | 值 |
|------|-----|
| **类型** | 教程区域 |
| **难度** | ★☆☆☆☆ |
| **主题** | 苏醒、遗忘 |
| **核心技能** | 翻滚、钩索 |
| **Boss** | 墓地守卫（教程Boss） |
| **设计目标** | 教学核心机制，建立氛围 |

**区域特色：**
- 线性但有分支的墓穴走廊
- 通过尸体和石碑进行环境叙事
- 渐进式难度：从空走廊→单个敌人→多个敌人→Boss
- 教学点分布在自然的游戏流程中，不打断体验
- 篝火位置合理，减少跑路时间

**隐藏内容：**
- 隐藏墓穴：需要钩索到达的高处平台，内有护符
- 密室：破坏特定墙壁可发现，内有灵魂道具
- NPC"无名者"：在隐藏角落遇到的第一个NPC，提供背景线索

**敌人配置：**
| 敌人 | 数量 | 分布 |
|------|------|------|
| 骷髅兵 | 5-7 | 走廊、战斗房 |
| 骷髅弓手 | 2-3 | 高处平台 |
| 墓地守卫 | 1 | Boss房 |

### 2.2 腐朽森林 (The Decaying Forest)
| 属性 | 值 |
|------|-----|
| **类型** | 探索向区域 |
| **难度** | ★★☆☆☆ |
| **主题** | 自然的腐化与重生 |
| **核心技能** | 灰烬回响 |
| **Boss** | 腐朽之母 |
| **设计目标** | 展示技能驱动探索 |

**区域特色：**
- 多层立体结构：地面层、树冠层、地下根系层
- 残影踩机关：某些平台需要灰烬回响的残影才能激活
- 腐化区域：进入会累积中毒值，需要快速通过或找到净化点
- 动态天气：偶尔会有灰烬风暴，降低视野
- 大量垂直探索，钩索是关键移动工具

**隐藏内容：**
- 树冠密室：需要连续使用钩索+残影才能到达
- 古老树洞：内有关于世界起源的石碑
- 腐朽之母的巢穴外围：发现关于Boss的背景故事

**敌人配置：**
| 敌人 | 数量 | 分布 |
|------|------|------|
| 腐化村民 | 8-10 | 地面层 |
| 骷髅兵 | 3-4 | 树根层 |
| 暗影猎手 | 1-2 | 树冠层（精英） |
| 腐朽之母 | 1 | Boss房 |

---

### 2.3 沉没教堂 (The Sunken Cathedral)
| 属性 | 值 |
|------|-----|
| **类型** | 战斗向区域 |
| **难度** | ★★★☆☆ |
| **主题** | 失落的信仰 |
| **核心技能** | 圣光术 |
| **Boss** | 堕落主教 |
| **设计目标** | 硬核战斗体验 |

**区域特色：**
- 黑暗区域：某些区域完全黑暗，需要圣光术照亮
- 水下区域：移动速度降低，但可以发现隐藏通道
- 信仰试炼：特定房间需要击败所有敌人才能通过
- 光影谜题：调整光线方向打开隐藏门
- 密集战斗：敌人数量多，考验战斗技巧

**隐藏内容：**
- 地下墓穴：需要圣光术照亮才能发现入口
- 主教的秘密书房：内有关于焚余教团的文献
- 被诅咒的圣器：可以解除诅咒获得强力装备

**敌人配置：**
| 敌人 | 数量 | 分布 |
|------|------|------|
| 骷髅兵 | 6-8 | 走廊、大厅 |
| 骷髅弓手 | 4-5 | 高处、黑暗区域 |
| 骷髅骑士 | 2-3 | 关键通道（精英） |
| 灰烬亡灵 | 1-2 | 黑暗区域（精英） |
| 堕落主教 | 1 | Boss房 |

### 2.4 灰烬矿坑 (The Ash Mines)
| 属性 | 值 |
|------|-----|
| **类型** | 技能向区域 |
| **难度** | ★★★☆☆ |
| **主题** | 挖掘与贪婪 |
| **核心技能** | 震地击 |
| **设计目标** | 技能与环境深度互动 |

**区域特色：**
- 可破坏地形：使用震地击破坏脆弱地面/墙壁发现隐藏区域
- 竖直矿井：大量垂直探索，需要钩索和震地击配合
- 矿车轨道：可以搭乘矿车快速移动（也需要震地击启动）
- 毒气区域：需要找到通风口或快速通过
- 机关密集：拉杆、压力板、可破坏墙壁

**隐藏内容：**
- 被掩埋的宝库：需要多次震地击才能到达
- 矿工的日志：揭示矿坑灾难的真相
- 蠕虫巢穴：发现矿坑蠕虫的起源

**敌人配置：**
| 敌人 | 数量 | 分布 |
|------|------|------|
| 骷髅兵 | 5-6 | 走廊 |
| 灰烬亡灵 | 3-4 | 毒气区域（精英） |
| 暗影猎手 | 1-2 | 暗处（精英） |
| 矿坑蠕虫 | 1 | Boss房 |

### 2.5 灰烬主城 (The Ash Capital)
| 属性 | 值 |
|------|-----|
| **类型** | 中心枢纽 |
| **难度** | 安全区 |
| **主题** | 残存的文明 |
| **核心技能** | 无（汇聚点） |
| **Boss** | 无 |
| **设计目标** | 安全感、社交、升级 |

**区域特色：**
- 安全区域，没有敌人
- 多层结构：地面层、地下层、塔楼
- NPC聚集地：所有服务NPC都在这里
- 随游戏进度解锁新区域和NPC对话
- 可以看到其他区域的远景

**NPC列表：**
| NPC | 功能 | 位置 | 说明 |
|-----|------|------|------|
| 铁匠 | 武器强化/修理 | 主城工坊 | 粗犷友善，记忆逐渐消退 |
| 商人 | 物品买卖 | 主城广场 | 神秘旅人，见多识广 |
| 先知 | 技能升级/洗点 | 主城塔楼 | 老年女性，知晓灰烬的秘密 |
| 神秘旅人 | 支线任务/线索 | 主城酒馆 | 沉默寡言，有自己的目的 |
| 守卫队长 | 区域信息 | 主城门口 | 提供各区域的敌人情报 |

### 2.6 王城废墟 (The Ruined Palace)
| 属性 | 值 |
|------|-----|
| **类型** | 中期区域 |
| **难度** | ★★★★☆ |

| **核心技能** | 血誓之刃 |
| **Boss** | 被放逐的国王 |
| **设计目标** | 血祭机制深度体验 |

**区域特色：**
- 血祭机关：需要消耗HP打开隐藏门/激活机关
- 曾经辉煌的宫殿，现在破败不堪
- 大量精英敌人，考验战斗技巧
- 王座室：大型Boss战场景
- 血池区域：进入会持续掉血

**隐藏内容：**
- 王室宝库：需要血祭才能打开

| 骷髅弓手 | 3-4 | 高处 |

### 2.7 深渊回廊 (The Abyss Corridor)
| 属性 | 值 |
|------|-----|
| **类型** | 中期区域 |
| **难度** | ★★★★☆ |
| **主题** | 虚空与黑暗 |
| **核心技能** | 暗影步 |
| **Boss** | 深渊守望者 |
| **设计目标** | 黑暗环境的紧张感 |

**区域特色：**
- 极度黑暗：视野极小，需要技能照明
- 空间扭曲：某些走廊会传送玩家到不同位置
- 暗影敌人：只能在光照范围内被攻击
- 悬崖和虚空：掉落即死
- 紧张的音效设计：回声、低语

**隐藏内容：**
- 深渊之眼：可以暂时照亮整个区域
- 虚空碎片：收集后解锁隐藏剧情
- 暗影守望者的起源故事

**敌人配置：**
| 敌人 | 数量 | 分布 |
|------|------|------|
| 暗影猎手 | 4-5 | 黑暗区域（精英） |
| 灰烬亡灵 | 2-3 | 走廊（精英） |
| 深渊守望者 | 1 | Boss房 |

| 属性 | 值 |

| **类型** | 中期区域 |

| **核心技能** | 火焰印记 |
| **Boss** | 火焰守护者 |
| **设计目标** | 火焰/冰霜环境互动 |

**区域特色：**

- 信仰试炼：特定房间需要完成挑战
- 温度机制：长时间在极端区域会受到持续伤害
- 祭坛互动：可以恢复状态或获得临时增益

**隐藏内容：**
- 古老火焰：永久提升火焰抗性
- 守护者的日记：揭示神殿的历史
- 隐藏试炼场：完成挑战获得火焰印记的升级
