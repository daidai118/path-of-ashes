## 一、任务粒度说明

### 任务标准

每个任务必须满足：
1. **单一职责** - 一个任务只做一件事
2. **可验证** - 有明确的完成标准
3. **可实现** - 1-4小时内可完成
4. **有依赖** - 明确前置任务

### 任务格式

```
任务ID: T-X.Y.Z
任务名称: [具体描述]
预计时间: [X]小时
前置依赖: [任务ID]
完成标准: [可验证的标准]
验收方法: [如何验证完成]
```

## 二、Phase 1: 垂直切片 (细粒度)

### Epic 1: 玩家系统

#### Story 1.1: 玩家移动系统

**T-1.1.1: 实现水平移动基础**
- 预计时间: 2小时
- 前置依赖: 无
- 完成标准:
  - 玩家可以左右移动
  - 有加速和减速过程
  - 移动速度符合设计值(200)
- 验收方法:
  - 运行游戏，按A/D键移动
  - 观察是否有加速过程
  - 测量移动速度

**T-1.1.2: 实现跳跃基础**
- 预计时间: 2小时
- 前置依赖: T-1.1.1
- 完成标准:
  - 按空格键跳跃
  - 跳跃高度符合设计值(-400)
  - 落地后可以再次跳跃
- 验收方法:
  - 运行游戏，按空格跳跃
  - 测量跳跃高度
  - 连续跳跃测试

**T-1.1.3: 实现土狼时间**
- 预计时间: 1小时
- 前置依赖: T-1.1.2
- 完成标准:
  - 离开平台后0.1秒内仍可跳跃
  - 超过0.1秒后无法跳跃
- 验收方法:
  - 走到平台边缘，离开后立即按空格
  - 测试是否可以跳跃
  - 延迟后测试是否无法跳跃

**T-1.1.4: 实现跳跃缓冲**
- 预计时间: 1小时
- 前置依赖: T-1.1.2
- 完成标准:
  - 落地前0.1秒按跳跃，落地后自动跳跃
  - 超过0.1秒后不触发
- 验收方法:
  - 跳跃中提前按空格
  - 观察落地后是否自动跳跃

**T-1.1.5: 实现翻滚**
- 预计时间: 2小时
- 前置依赖: T-1.1.1
- 完成标准:
  - 按Shift键翻滚
  - 翻滚速度400
  - 翻滚持续0.3秒
  - 翻滚后0.5秒冷却
  - 翻滚期间无敌
- 验收方法:
  - 按Shift翻滚
  - 测量翻滚距离
  - 翻滚期间让敌人攻击测试无敌

**T-1.1.6: 实现朝向切换**
- 预计时间: 0.5小时
- 前置依赖: T-1.1.1
- 完成标准:
  - 向左移动时精灵水平翻转
  - 向右移动时精灵恢复正常
- 验收方法:
  - 左右移动观察精灵方向

**T-1.1.7: 实现动画状态机**
- 预计时间: 2小时
- 前置依赖: T-1.1.6
- 完成标准:

**T-1.2.5: 实现受击判定(Hurtbox)**
- 预计时间: 1.5小时

  - 受伤时播放hurt动画

  - 观察是否扣血
  - 观察无敌时间

**T-1.2.6: 实现体力系统**
- 预计时间: 2小时
- 前置依赖: T-1.2.1
- 完成标准:
  - 初始体力100
  - 轻攻击消耗15
  - 重攻击消耗30
  - 翻滚消耗25
  - 脱战恢复30/秒
  - 战斗恢复15/秒
  - 体力耗尽3秒无法恢复
- 验收方法:
  - 连续攻击观察体力消耗
  - 等待观察恢复速度
  - 体力耗尽后测试惩罚

**T-1.2.7: 实现伤害计算**
- 预计时间: 1.5小时
- 前置依赖: T-1.2.4
- 完成标准:
  - 基础伤害 = 武器伤害 × (1 + 力量%)
  - 防御减伤 = 防御 / (防御 + 100)
  - 最终伤害 = 基础伤害 × (1 - 防御减伤)
- 验收方法:
  - 打印伤害计算过程
  - 验证公式正确性
