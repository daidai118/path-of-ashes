# 翻滚残影特效 (Roll Afterimage VFX)

## 基本信息
- **类型**: vfx
- **用途**: 翻滚时的残影特效
- **尺寸**: 64x64
- **动画帧数**: 4帧
- **输出格式**: PNG, sprite sheet（横向排列，64x256）

## 风格标准
> Dark fantasy pixel art, 16-bit retro style, gloomy medieval atmosphere, muted desaturated colors with warm accent lighting (amber/orange from fire), high contrast shadows, hand-crafted pixel detail.

## 提示词 (Prompt)

### 正向提示词
```
Dark fantasy pixel art VFX sprite sheet, 16-bit retro style, roll afterimage effect, 4 frames horizontal sprite sheet, 64x64 per frame, grey semi-transparent ghost trail following character, dark souls style dodge trail, afterimage appearing then fading, transparent background, game VFX asset, pixel perfect, motion blur trail with ash particles, muted grey to transparent gradient
```

## 参考说明
- 风格参考: Dead Cells的翻滚特效
- 动作: 残影出现→跟随移动→消散
- 关键帧: 出现(1帧)→跟随(1帧)→消散(2帧)

## 备注
- 残影应是角色的半透明轮廓
- 颜色：灰色→透明
- 可以有轻微的灰烬粒子效果
